# To Do List App

A simple command-line To Do List application built with Python.

## Features

* Add new tasks
* View all tasks
* Delete tasks
* Save tasks automatically using JSON

## Technologies Used

* Python
* JSON File Handling

## How to Run

1. Make sure Python is installed on your computer.
2. Open the project folder in VS Code.
3. Run the following command in the terminal:

```bash
python PYTHON.py
```

## Menu Options

* Add task
* List tasks
* Delete task
* Exit program

## Project Structure

* `PYTHON.py` → Main application file
* `tasks.json` → Stores saved tasks

## Notes

Tasks are saved automatically in `tasks.json`, so they remain available after closing the program.
